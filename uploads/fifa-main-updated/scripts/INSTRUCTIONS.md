# Стъпка по стъпка – Настройка на автоматични резултати

## Стъпка 0: Разархивирай проекта

Разархивирай целия `fifa-main-api.zip` в папка по твой избор, напр.:

```
C:\fifa-main\
```

Вътре трябва да имаш:
```
C:\fifa-main\
  index.html
  app.js
  styles.css
  results.json
  data\results.json
  scripts\
    .env                  <- ключът ти вече е тук, готов
    update-results.js
    update-results.bat
    README.md
```

## Стъпка 1: Инсталирай Node.js (ако нямаш)

1. Отиди на https://nodejs.org
2. Изтегли и инсталирай **LTS версията** (зелен бутон)
3. Следвай инсталатора с "Next" -> "Next" -> "Install" (по подразбиране)
4. Рестартирай компютъра ако те пита

## Стъпка 2: Тествай ръчно

1. Отвори папка `C:\fifa-main\scripts`
2. Двукратно кликни на `update-results.bat`
3. Ще се отвори черен прозорец (команден ред) с текст:
   ```
   Тегля фикстури от api-football.com...
   Получени 104 мача от API.
   Намерени резултати за X от нашите мачове.
   Записани: results.json и data/results.json
   ```
4. Ако видиш това – работи! Ако видиш грешка, виж секция "Проблеми" по-долу.

⚠️ Ключът ти вече е сложен в `.env` файла – не трябва да правиш нищо допълнително тук.

## Стъпка 3: Провери приложението

1. Двукратно кликни на `start-fifa-world-2026.bat` (в главната папка `C:\fifa-main\`)
2. Отвори се браузър на `http://127.0.0.1:8026/`
3. Иди на таб "Мачове" или "Класиране" – трябва да видиш реалните резултати

## Стъпка 4: Автоматично обновяване на всеки 30 минути (Task Scheduler)

1. Натисни **Win** клавиш, напиши `Task Scheduler`, отвори го
2. В дясната колона кликни **Create Task...** (Създай задача)
3. **General таб:**
   - Name: `FIFA Results Update`
   - Избери "Run whether user is logged on or not" (по желание)
4. **Triggers таб:**
   - Кликни **New...**
   - Begin the task: `On a schedule`
   - Settings: `Daily`
   - Recur every: `1 days`
   - Кликни **Advanced settings** -> отбележи `Repeat task every` -> избери `30 minutes`
   - For a duration of: `Indefinitely`
   - Натисни **OK**
5. **Actions таб:**
   - Кликни **New...**
   - Action: `Start a program`
   - Program/script: натисни **Browse** и избери:
     ```
     C:\fifa-main\scripts\update-results.bat
     ```
   - Start in (optional):
     ```
     C:\fifa-main\scripts
     ```
   - Натисни **OK**
6. **Conditions / Settings табове:** остави по подразбиране
7. Натисни **OK** за да запазиш задачата
8. Ако те пита за парола на Windows акаунт – въведи я (или избери "Run only when user is logged on" ако не искаш парола)

### Проверка дали работи:
- В Task Scheduler library, намери `FIFA Results Update`
- Кликни десен бутон -> **Run** (за тест веднага)
- Провери `Last Run Result` -> трябва да бъде `0x0` (успех = OK)

## Готово!

Сега `results.json` ще се обновява автоматично на всеки 30 минути, а отвореното приложение
вече само презарежда тези данни на всеки 60 секунди. Не трябва да пипаш нищо повече.

## Проблеми

**"node не е разпознат като команда"**
-> Node.js не е инсталиран или трябва рестарт на компютъра след инсталация.

**"ГРЕШКА: Липсва API ключ"**
-> провери дали файл `scripts\.env` съществува и съдържа реалния ключ
(не "твоят_ключ_тук").

**"Намерени резултати за 0 мача"**
-> възможно е Световното още да не е стартирало мачовете, или `LEAGUE_ID`/`SEASON`
да не съвпадат. Това е нормално преди 11 юни 2026.

**Искам да спра автоматичното обновяване**
-> Task Scheduler -> намери `FIFA Results Update` -> десен бутон -> **Disable** или **Delete**

**Лимит на безплатния план**
-> 100 заявки/ден. На всеки 30 мин = 48 заявки/ден. Безопасно.
Ако искаш по-често (напр. на 15 мин = 96/ден), е на ръба на лимита – не препоръчвам.
