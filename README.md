# FIFA WORLD 2026 — LIVE

Готово web приложение за Световното първенство 2026 с автоматични резултати на живо.

## Стартиране

Отвори `start-fifa-world-2026.bat`. Страницата ще се зареди на:

`http://127.0.0.1:8026/`

## Данни на живо

- резултатите идват директно от публичния ESPN World Cup endpoint;
- двойките, домакин/гост, датите, часовете и стадионите също идват от ESPN;
- не е нужен API ключ или регистрация;
- обновяването е автоматично на всеки 60 секунди;
- статусът горе показва `НА ЖИВО` и часа на последното успешно обновяване;
- при временен проблем с ESPN приложението използва резервния `results.json`.

Приложението изчислява автоматично класирането, прогреса, головете и елиминационната схема. Резултатите не могат да се редактират ръчно.

## GitHub Pages

Качи съдържанието на тази папка в основната папка на GitHub repository. Включеният workflow обновява резервния JSON на всеки 5 минути. Самата страница продължава да получава директни live данни на всеки 60 секунди.
