# FIFA WORLD 2026

Самостоятелно web приложение за Световното първенство 2026.

## Какво включва

- 12 групи с по 4 отбора;
- групови мачове кой срещу кого играе;
- автоматично зареждане на резултати от `data/results.json`;
- заключени полета за резултати, без ръчно попълване;
- автоматично класиране по точки, голова разлика и отбелязани голове;
- общ прогрес на груповата фаза;
- изглед за елиминационната фаза;
- запазване на въведените резултати в браузъра.

## Стартиране

Най-лесно: стартирай `start-fifa-world-2026.bat`.

Приложението ще се отвори на:

`http://127.0.0.1:8026/`

## Бележка

Приложението обновява резултатите от `data/results.json` на всеки 60 секунди. Ако искаш истински live резултати от външен доставчик, трябва да се свърже конкретен sports API или собствен backend, който обновява този JSON файл.

Пример за резултат:

```json
{
  "updatedAt": "2026-06-12T12:00:00Z",
  "matches": {
    "A-1": { "home": 2, "away": 1 },
    "A-2": { "home": 0, "away": 3 }
  }
}
```
